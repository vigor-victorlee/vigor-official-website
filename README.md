# VIGOR Official Website

Marketing site for VIGOR — operator-built deeptech across manufacturing, healthcare, government, smart city, retail, and immersive.

Static HTML/CSS/JS implementation based on the Claude Design handoff bundle.

## Structure

- `index.html` — home page
- `css/tokens.css`, `css/site.css` — design tokens and site styles
- `js/megamenu.js`, `js/tweaks.js` — navigation mega menu and runtime tweaks
- `assets/` — logo SVGs

## Local preview

Serve the directory with any static server:

```
python3 -m http.server 8000
```

Then open http://localhost:8000.
